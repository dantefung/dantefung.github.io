2018.03.26 dantefung  热度 ℃

创建项目

1、打开终端，键入命令

npm init -y

该命令可以初始化一个项目描述文件package.json文件

cnpm install vue –save

默认就会安装最新版本

问题

1、如果环境遇到“cnpm：command not found”

问题是因为你没有安装淘宝的镜像,在终端键入如下命令即可:

npm install cnpm -g –registry=https://registry.npm.taobao.org
